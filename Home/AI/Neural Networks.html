Frequently Asked Questions

What is a Neural Network?

A neural network is a series of algorithms that endeavors to recognize underlying relationships in a set of data through a process that mimics the way the human brain operates.

When should I use CNN vs. RNN?

Use Convolutional Neural Networks (CNN) for spatial data like images. Use Recurrent Neural Networks (RNN) for sequential data like text or time-series.

Can you optimize a network for mobile devices?

Yes, our experts use techniques like quantization and pruning to make large neural networks run efficiently on mobile and IoT hardware.
